TypeScript - 1

2019/12/29

type script에 대해 알아보자 (1) - 설치

NPM 으로 타입스크립트 설치

폴더를 하나 만들고 npm init 후 typescript 모듈 설치

package.json 의 scripts 항목에 typescript를 실행하는 스크립트 추가

src 폴더를 만들고 index.ts 파일을 추가

앞으로 추가되는 모든 ts파일들은 이 폴더에 생성한다.

아래의 명령어로 typescript를 javascript 로 transpiling 할 수 있다.

  • --outFile 옵션으로 결과물로 나온 파일의 위치, 이름을 지정가능
  • src/index.ts 는 내가 transpiling 할 typescript 파일

build 폴더에 다음과같은 결과물 확인가능

별도의 config 파일을 통해 설정값을 줄 수 있다.

tsconfig.json 파일이 생기는데 여기다가 다양한 설정가능

단순히 npx tsc 라고 터미널에 입력하면 프로젝트의 모든 .ts 파일을 같은 위치에 같은 이름으로 .js 파일로 변환하여 생성해준다.

tsconfig.json 에 outDir을 통해 결과물을 저장할 폴더를 지정할 수 있다.

package.json에 script 추가

이제 터미널에 npm run build-ts 하면 bundle 폴더에 결과물이 나올것이다.

만약 다음과 같은 에러가 나온다면

tsconfig.json에 "lib": ["es2015"],를 추가하자

다음과 같은 에러가 나온다면

tsconfig.json에 "lib": ["es2015", "dom"],를 추가하자

bundle 폴더에 결과물이 나온걸 확인할 수 있다.

npm run start 로 결과를 보자.
