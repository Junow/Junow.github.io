2019/12/29 카테고리 web

type script에 대해 알아보자 (3) - type

타입스크립트는 사용자가 자바스크립에서 기대하는것과 같은 타입들(숫자, 문자열, 구조, 부울) 을 지원한다.

​ 타입 선언을 통해 개발자가 코드를 예측가능할 수 있도록 도와주고 문법 에러나 타입과 일치하지 않는 값의 할당 등 기본적인 오류를 런타임 이전에 찾아낼 수 있다.

Boolean

Number

String

  • template string 을 이용해서도 물론 가능

Array

  • 두가지 방식이 있다.
  • 두번째는 제네릭 배열 타입 사용₩

Tuple

  • 튜플은 고정된 개수의 요소 타입을 알고 있지만 반드시 같을 필요는 없는 배열을 나타낼 때 사용

Enum

Any

  • 해당값의 타입을 모를경우 사용가능
  • 너무 남발하면 코드 분석에 타입체의 도움은 받을 수 없으므로 주의

Void

  • any 와 비슷하지만 어떤 타입의 부재도 전혀 없다.
  • 일반적으로 반환 값이 없는 함수의 반환타입으로 쓰일 수 있다.
  • 하지만 void 타입의 변수는 undefined 만 할당할 수 있으므로 유용하지 않다. (null 도 가능하다고 하나 안됨)
